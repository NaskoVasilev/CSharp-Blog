TODO List using ASP.NET Core Mvc and .Net Core 2.2

Today I will show you how to create a simple web application using c#!

Ok! Let's start! It will be fun!!!

Create new Project in Visual Studio
  • Open Visul Studio 2017
  • Go to File -> New Project
  • Choose .NET Core
  • Change authentication to individual user accounts

Now we have new project with authentication out of the box! We can login and register now! Try the functionality and let's continue then!

Configure database
  • Open appsettings json file in the root folder and change the connection string!
  • Then in folder Data create new folder Models and in it create class Task.

Open Paskage Manager Console and press the folowing commadnds:

  1. add-migration AddTaskEntity
  2. update-database

Now we have our database created! Open SSMS and see if everything is OK!!!
